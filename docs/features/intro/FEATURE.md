# 인트로 페이지

> 작성: 2025-11-21
> 상태: 기획 완료, 일러스트 대기 (캐릭터 선정 후 진행)

## 무엇을 만드는가

앱 첫 실행 시 MiniLine의 핵심 가치(Fragment → AI 조합 → Draft)를 3개 화면으로 설명하는 온보딩 페이지.

## 유저 시나리오

### 주요 플로우

1. 사용자가 앱을 처음 설치하고 실행
2. 인트로 페이지 3장이 순서대로 표시됨
3. 좌우 스와이프로 화면 전환
4. 마지막 화면에서 "시작하기" 버튼 클릭
5. 로그인/회원가입 화면으로 이동
6. 이후 앱 실행 시 인트로 건너뛰고 바로 메인 화면

### 엣지 케이스

- **건너뛰기**: 화면 1, 2에서 "건너뛰기" 버튼 → 바로 로그인 화면으로 이동
- **앱 종료 후 재실행**: 인트로 완료 전 종료 → 다시 인트로부터 시작
- **재설치**: SharedPreferences 초기화 → 인트로 다시 표시

## 화면 구성

### 레이아웃

```
┌─────────────────────────────────┐
│                                 │
│        [일러스트 영역]           │  ← 상단 60%
│                                 │
├─────────────────────────────────┤
│     [제목]                      │  ← 하단 40%
│     [설명]                      │
│     ● ○ ○                       │  ← 페이지 인디케이터
│     [시작하기]  (마지막만)       │
└─────────────────────────────────┘
```

### 화면 1: 문제 상황

- **제목:** 생각을 기록하고 싶은데...
- **설명:** 일기는 부담스럽고 / 메모는 어디 갔는지 모르겠고
- **일러스트:** 흩어진 노트, 메모장, 일기장 (회색 톤)

### 화면 2: 해결책

- **제목:** 그냥 짧게 적기만 하면 돼요
- **설명:** "점심에 파스타 먹음" / "친구가 추천한 식당이었는데" / 언제 어디서든
- **일러스트:** 스마트폰에서 짧은 문장 입력 (파스텔 톤)

### 화면 3: 결과

- **제목:** AI가 하나의 이야기로 만들어줘요
- **설명:** 시간을 넘어 자동으로 연결 / 완성된 글로 조합
- **일러스트:** 조각들이 별자리처럼 연결 (골드 톤)
- **버튼:** [MiniLine 시작하기]

## 일러스트 (대기)

캐릭터/마스코트 선정 후 미드저니 v7로 제작 예정.

**스타일 가이드:**
- 미니멀리스트 일러스트레이션
- 소프트 파스텔, 부드러운 그라데이션
- 비율: 9:16 (모바일 세로)
- 색상 진행: 회색 → 파스텔 → 골드

**프롬프트 초안:** [IMPLEMENTATION.md](./IMPLEMENTATION.md) 참조

## E2E 테스트

### 정상 플로우

- [ ] 첫 실행 → 인트로 표시 → 3장 스와이프 → 시작하기 → 로그인 화면
- [ ] 인트로 완료 후 재실행 → 바로 메인 화면

### 예외 시나리오

- [ ] 화면 1에서 건너뛰기 → 로그인 화면
- [ ] 인트로 중 앱 종료 → 재실행 시 인트로 처음부터
- [ ] 앱 재설치 → 인트로 다시 표시
