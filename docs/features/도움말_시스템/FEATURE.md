# 도움말 시스템

> Draft/Post 관련 사용자 안내 및 구독 유도

## 무엇을 만드는가

사용자가 Draft와 Post 기능을 이해하고 활용할 수 있도록 컨텍스트 기반 도움말 제공. 무료 한도 표시 및 프리미엄 유도를 통한 구독 전환 촉진.

## 유저 시나리오

### Empty State (빈 화면)

1. **Draft 탭 첫 방문 (비어있음)**
   - 원형 아이콘 배경 + sparkles 아이콘
   - "AI가 초안을 만들어요"
   - "토막글을 쌓으면 AI가 관련된 내용을 자동으로 묶어 초안을 제안합니다"
   - 힌트 박스: "토막글이 충분히 쌓이면 AI가 자동으로 관련 내용을 묶어 제안해요"

2. **Post 탭 첫 방문 (비어있음)**
   - 원형 아이콘 배경 + posts 아이콘
   - "완성된 글이 여기에"
   - "초안을 수락하면 AI가 완성된 글로 만들어드려요"
   - 힌트 박스: "초안을 수락하고 템플릿을 선택하면 AI가 완성된 글을 만들어요"

### 한도 배너 (PostLimitBanner)

1. **무료 사용자 + Post 1개 이상 생성**
   - Posts 탭 상단에 배너 표시
   - "이번 달 N/3개 사용" + 리셋 날짜

2. **한도 임박 (2/3 사용)**
   - primary 색상 강조
   - 동일 텍스트

3. **한도 초과 (3/3 사용)**
   - destructive 색상
   - "프리미엄으로 업그레이드" 링크 추가

### 프리미엄 시트 (PremiumSheet)

1. **트리거**
   - 한도 배너에서 "업그레이드" 탭
   - Post 생성 시 한도 초과

2. **내용**
   - 무료 vs 프리미엄 비교 테이블 (글 생성, AI 분석, 스토리지)
   - 프리미엄 기능 목록 (무제한, 실시간, 10GB, 우선 지원)
   - 업그레이드 버튼 (인앱결제 연결 예정)
   - 구매 복원 버튼

## 논리 흐름

```
Posts 탭 진입
├── 프리미엄 사용자 → 배너 숨김
└── 무료 사용자
    ├── freePostsCount == 0 → 배너 숨김
    └── freePostsCount > 0 → 배너 표시
        ├── count < 2 → 기본 스타일
        ├── count == 2 → 경고 스타일 (primary)
        └── count >= 3 → 초과 스타일 (destructive) + 업그레이드 링크
```
